<p>Antes de fazer qualquer alteração no sistema operacional ou no hardware de seu #computador, é sempre recomendável fazer um #backup dos principais arquivos para que eles não sejam perdidos em casos de acidentes. Infelizmente, nem sempre é possível fazer esta tarefa, mas não há motivo para desespero, já que eles ainda podem ser recuperados em situações deste tipo. #TESTE #TESTE #TESTE @TESTE </p><p>Para isto, uma das soluções mais práticas e eficazes consiste em utilizar o #Linux para acessar o seu #HD e salvar os arquivos que estão contidos nele. A seguir, veja como utilizar uma distribuição do Linux para salvar os arquivos de um computador que não está conseguindo inicializar o Windows:</p><p><br></p><p><strong>Importante</strong></p><p><br></p><p><strong>Veja também:</strong></p><p><a href="https://olhardigital.com.br/dicas_e_tutoriais/noticia/diario-de-bordo-como-ressuscitar-o-seu-pc-de-2004-com-o-linux-em-2018/79987" target="_blank">Diário de bordo: Como ressuscitar o seu PC de 2004 com o Linux em 2018</a></p><p><a href="https://olhardigital.com.br/dicas_e_tutoriais/noticia/guia-do-linux-mint-aprenda-a-instalar-programas-no-sistema/79105" target="_blank">Guia do Linux Mint: Aprenda a instalar programas no sistema</a><a href="https://olhardigital.com.br/dicas_e_tutoriais/noticia/15-programas-do-windows-que-tambem-rodam-no-linux/78285" target="_blank">15 programas do Windows que também rodam no Linux</a></p><p><a href="https://olhardigital.com.br/dicas_e_tutoriais/noticia/dicionario-linux-entenda-10-termos-importantes-do-sistema-operacional/77931" target="_blank">Dicionário Linux: entenda 10 termos importantes do sistema operacional</a></p><p><br></p><p>Antes de prosseguir, é preciso explicar que a forma que será exibida abaixo não fará a instalação do Linux no computador. Assim, o Linux será rodado direto a partir de um pen-drive e, após salvar os seus arquivos, você poderá prosseguir para formatar o computador e reinstalar o Windows nele.</p><p>Além destas ressalvas, também é preciso dizer os itens necessários para realizar esta tarefa:</p><ul><li>Pendrive com 4GB ou mais de espaço para armazenamento para a distribuição Linux. Alternativamente, você pode usar um DVD-R, caso possua um gravador de DVD para gravar a imagem ISO;</li><li>Um segundo pendrive ou HD externo para o backup dos arquivos.</li></ul><p><br></p><p><strong>Preparando o pendrive ou DVD-R</strong></p><p><br></p><p>Para preparar um pendrive, a forma mais fácil é ter o acesso a uma máquina que esteja funcionando com o Windows instalado. Para este tutorial, a distribuição escolhida foi o Linux MInt com o ambiente gráfico do XFCE. Veja como é feito o processo:</p><ol><li>Acesse o site do Linux Mint por este <a href="https://www.linuxmint.com/download.php" target="_blank">link</a> e realize o seu download. Caso esteja em dúvida sobre o seu processador, baixe a versão 32-bit;</li><li><br></li><li><img src="https://cache.olhardigital.com.br/uploads/acervo_imagens/2018/12/20181211121200.jpg" alt="Reprodução"></li><li>Feito o download da distribuição, é necessário baixar uma ferramenta extra para colocá-la em um pendrive. O Rufus é um dos programas mais recomendados para isto, então, realize o download dele por este <a href="https://rufus.ie/en_IE.html" target="_blank">link</a>. Ao fim do download, execute-o. O programa não requer instalação;</li><li><br></li><li><img src="https://cache.olhardigital.com.br/uploads/acervo_imagens/2018/12/20181211121254.jpg" alt="Reprodução"></li><li>Em “Dispostivo”, escolha a unidade de pendrive que está na máquina. Agora, clique em “Selecionar” e escolha o arquivo ISO do Linux Mint que você baixou. Por fim, clique em “Iniciar”.</li><li><br></li><li><img src="https://cache.olhardigital.com.br/uploads/acervo_imagens/2018/12/20181211121446.jpg" alt="Reprodução"></li></ol><p>O processo para passar a imagem ISO para um pendrive varia de acordo com a velocidade de seu dispositivo. Uma vez que o processo está feito, você pode prosseguir para o computador que tem os arquivos a serem salvos.</p><p><strong>Recuperando os arquivos com o Linux</strong></p><p>Antes de começar a recuperar os arquivos da máquina, primeiro é necessário fazer com que o computador inicie pelo Pendrive. Para isto, é necessário realizar uma alteração em sua BIOS. Veja:</p><ol><li>Coloque o pendrive no computador e inicie-o apertando a tecla F2 repetidamente para entrar na BIOS. Dependendo da fabricante de sua placa mãe, a tecla usada e a própria BIOS pode ter uma aparência diferente do que está ilustrado abaixo;</li><li>Em General, entre em “Boot Sequence” e coloque o “USB Device” como o primeiro item da lista. Nesta parte, geralmente são utilizados os botões de “Page Up/Down”. Em BIOS mais recentes, você deve estar apto a utilizar o mouse;</li><li><br></li><li><img src="https://cache.olhardigital.com.br/uploads/acervo_imagens/2018/12/20181211121512.jpg" alt="Reprodução"></li><li>Saia da BIOS e salve as alterações caso solicitado.</li></ol><p>Neste momento, dependendo do seu computador, pode ser que o Linux Mint lhe apresente uma tela perguntando o que você deseja fazer. Se isto lhe ocorrer, escolha a opção para “Testar o Linux Mint sem instalá-lo”, caso contrário, a máquina já fará o boot no Linux diretamente ainda sem realizar a sua nstalação.</p><p>Uma vez dentro do sistema, o processo para recuperar seus arquivos é bem simples. Então, siga estes passos para isto:</p><ol><li>Conecte o pendrive ou HD externo que receberá o backup ao computador;</li><li>Na “Área de trabalho”, dê um clique duplo em “Home”;</li><li><br></li><li><img src="https://cache.olhardigital.com.br/uploads/acervo_imagens/2018/12/20181211121541.jpg" alt="Reprodução"></li><li>Já em “Devices”, escolha o item que mostra o tamanho do HD que está instalado em seu computador;</li><li><br></li><li><img src="https://cache.olhardigital.com.br/uploads/acervo_imagens/2018/12/20181211121611.jpg" alt="Reprodução"></li><li>Neste momento, você já deve ver os arquivos que estavam em seu computador, que estão prontos para serem copiados ao pendrive.</li></ol><p>Como um exemplo para esta matéria, será exibido o processo para recuperar os documentos que ficam na pasta “Meus documentos” do Windows, uma vez que estes arquivos ficam um pouco mais escondidos. Veja:</p><ol><li>Ainda na tela do passo anterior, dê um clique duplo em “Users”;</li><li><br></li><li><img src="https://cache.olhardigital.com.br/uploads/acervo_imagens/2018/12/20181211121642.jpg" alt="Reprodução"></li><li>Procure pelo seu nome de usuário do Windows e dê um clique duplo em sua pasta;</li><li><br></li><li><img src="https://cache.olhardigital.com.br/uploads/acervo_imagens/2018/12/20181211121716.jpg" alt="Reprodução"></li><li>Você agora verá as pastas “Desktop”, que é da “Área de trabalho”, a “Documents”, “Meus Documentos” e a “Downloads”, que é onde ficam os seus principais arquivos;</li><li><br></li><li><img src="https://cache.olhardigital.com.br/uploads/acervo_imagens/2018/12/20181211121744.jpg" alt="Reprodução"></li><li>Agora, basta selecionar o que deseja, dar um clique duplo com o botão direito do mouse neles e usar a opção “Send to” escolhendo o pendrive ou o HD externo que receberá o backup.</li><li><br></li><li><img src="https://cache.olhardigital.com.br/uploads/acervo_imagens/2018/12/20181211121811.jpg" alt="Reprodução" width="860" height="538"></li></ol><p>Pronto! Agora, tudo o que você precisa fazer é reinstalar o Windows e, então, copiar os seus arquivos de volta para ele, uma vez que o Linux não mudará o formato de partição utilizado pelo pendrive.</p>
